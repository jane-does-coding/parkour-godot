# Parkour Platformer! (Or an attempt at it)

Itch: https://yevheniiasimakagmailcom.itch.io/parkour-godot

This is a 2nd ever game I've made with Godot. This time I've learned to use layers for the game map. And also signals, and how to connect things. I did all things here on my own, and tested what I know. I discovered tilesets and charactrer on itch.io, and learned how to do that as well.

Installation

Clone the repository

```
git clone https://github.com/jane-does-oding/parkour-godot
```

1. Install Godot Engine
2. Download the latest stable version of Godot Engine
3. Open the project in Godot
4. Launch Godot.
5. Click “Open Project” and navigate to the folder where you cloned or extracted the project.
6. Select the folder and click “Open”.
7. Run the project
